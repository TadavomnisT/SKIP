# SKIP

![SKIP Logo](./docs/SKIP_logo.png)

S.K.I.P: Secure Key-logger for GNU/Linux using PHP


## Installation

```Shell
git clone https://github.com/TadavomnisT/SKIP.git
cd SKIP/src
```
or
```shell
wget https://raw.githubusercontent.com/TadavomnisT/SKIP/main/src/SKIP.php
```

## Usage

```shell
sudo php SKIP.php
```
